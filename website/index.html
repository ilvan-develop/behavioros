YAML Configuration

One File. Full Organization.

Define your entire AI team's behavior, governance, and quality standards in a single behavioros.yaml.

behavioros.yaml 
# behavioros.yaml
id: my-team
name: My AI Team

personas:
  - role: engineer
    authority: senior
  - role: qa
    authority: senior

governance:
  - name: Code Review Required
    level: high
    action: block

quality:
  - name: Test Coverage
    type: test_coverage
    threshold: 80
